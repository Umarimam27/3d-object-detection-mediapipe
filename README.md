# 3d-object-detection-mediapipe
A simple Streamlit app for 3D object detection using MediaPipe Objectron. Upload an image and detect Cup, Shoe, Chair, or Camera with 3D bounding boxes and axis visualization. Clean UI, fast inference, and perfect for learning 3D vision with MediaPipe.
